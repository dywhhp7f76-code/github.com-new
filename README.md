## 📥 Как внести вклад

1. **Форкните** репозиторий
2. **Клонируйте** ваш форк: `git clone https://github.com/your-username/github.com-new.git`
3. **Добавьте** удалённый репозиторий: `git remote add upstream https://github.com/dywhhp7f76-code/github.com-new.git`
4. **Создайте ветку**: `git checkout -b feature/your-feature`
5. **Внесите изменения** и закоммитьте
6. **Запушьте** в ваш форк: `git push origin feature/your-feature`
7. **Откройте Pull Request** на GitHub
